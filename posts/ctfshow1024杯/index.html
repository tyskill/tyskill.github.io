FastAPI 介绍：
      FastAPI是一个Python编写的高性能web框架。
      FastAPI这个Python Web框架并没有带渲染的网页模板引擎

看到Python就想到SSTI，接下来寻找利用点，扫一波目录，发现/docs路由，POST传值访问/cccalccc，回显Json数据：

res: 1
err: false

尝试常规SSTI套路，发现{{}}{%%}都会让err为True，而正常输入字符串却不会，这里确实不知道渲染模板引擎是啥，所以可能是eval或者exec等函数的命令执行。

但是在使用"".__class__时又会引起Internal Server Error，可能是因为数据类型的原因，使用函数str转换类型str("".__class__)，果然可以。

import zipfile, os
from itertools import permutations
# 排列组合
passwd, password = list(permutations('0124', 4)), []
for i in passwd: password.append(''.join(i))
print("[*]Password is ready...\n[*]Brute-force starts...")
# 爆破
file = '1024.zip'
for i in range(1024):
# while True:
    zfile = zipfile.ZipFile(file)
    for p in password:
        try:
            zfile.extractall(members=zfile.namelist(), pwd=p.encode('utf-8'))
            zfile.close()
            try:
                os.remove(file)
            except OSError as e:
                print(e)
            files = os.listdir()
            # print(files)
            for f in files:
                if f.endswith('.zip') and f != file:
                    file = f
                    break
            break
        except:
            pass
    print("[*]unzip " + file)

1024.txt内容解码：

hex + hex + b64 + b32 + hex + b64 + hex + b64 + hex + b32 + b32 + hex + b64 + unzip

得到flag。
